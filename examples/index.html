placeholder

An experiment exploring different methods of creating placeholders for images in our fresh interactive wireframes we're all using these days. There are services that will generate images, but, then you're depending on an internet connection. Hopefully, you will find one of these methods suitable and lightweight for what you're looking for.

Methods:

  • SASS mixin
  • Vanilla Javascript yay, no dependencies
  • Polymer fancy custom element based on Google's polymer project

In all the methods there are the basic concepts of creating a container that gives flexible ways to define the dimensions. The following combinations can be used to define the layout.

  • width and height
  • width and ratio
  • ratio

The advantage of using the ratio is that you can maintain an aspect ratio of the type of image you will be using. Also, if the width is set in px and the height as a %, the height percentage will be used as ratio in terms of the width, not of the parent.

ie: width = 200px, height = 25%, the container should result in a height (based on the ratio 0.25) of 50px

sass mixin

The idea behind the sass mixin is that you would use the mixin on css selectors to create placeholders. 

@include placeholder($width: null, $height: null, $ratio: null, $options: (use-ratio-height: null))

examples

#placeholder {
	@include placeholder(250px, 320px);
}

#placeholder {
	@include placeholder(60%, 25%);
}

#placeholder {
	@include placeholder(400px, 60%);
}

#placeholder {
	@include placeholder(30%, 420px);
}

#placeholder {
	@include placeholder(null, null, 0.3);
}

#placeholder {
	@include placeholder(320px, null, 0.8);
}

#placeholder {
	@include placeholder(60%, null, 0.45);	
}

vanilla js

Create placeholders with good 'ol javascript, no dependencies on any other library. Create an element, or select one that already exists, and pass in the DOM element with the necssary options. This is method is called Placeholder Vanilla because there are plans to write a Placeholder jQuery, too. 

new PlaceholderVanilla(element, options)

examples

new PlaceholderVanilla(element, {
	width: '450px',
	height: '235px'
}).applyDimensions();

new PlaceholderVanilla(element, {
	width: '30%',
	height: '150%'
}).applyDimensions();

new PlaceholderVanilla(element, {
	width: '600px',
	height: '50%'
}).applyDimensions();

new PlaceholderVanilla(element, {
	width: '80%',
	height: '200px'
}).applyDimensions();

new PlaceholderVanilla(element, {
	ratio: 0.35
}).applyDimensions();

new PlaceholderVanilla(element, {
	width: '350px',
	ratio: 1.25
}).applyDimensions();

new PlaceholderVanilla(element_ratio_with_width_per, {
	width: '45%',
	ratio: 1
}).applyDimensions();
